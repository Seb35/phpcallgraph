PHPCallGraph - A Static Call Graph Generator for PHP

PHPCallGraph is a tool to generate static call graphs for PHP source code. Such a graph shows the call dependencies between methods or functions of an application, i.e. the methods which called by a certain method.

The call graph generator uses the CodeAnalyzer of the InstantSVC project and DOT. The graphs can be leveraged to gain a better understanding of large software systems or even to debunk design flaws in them.
